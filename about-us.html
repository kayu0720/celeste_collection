OUR STORY

關於Celeste Collection

隨著黑膠唱片與實體音樂收藏文化逐漸受到重視，我們希望打造一個提供唱片搜尋、購買、收藏與評論功能的平台，讓音樂愛好者能夠更方便地找到自己喜愛的專輯，同時推廣唱片收藏文化，提升使用者的購物與收藏體驗。

核心團隊成員

郭曉螢

網站程式功能設計與修改

本次期末專案是這學期網頁設計課程的集大成。從最初的版面規劃到最終的網頁呈現，我深刻體會到 HTML 做為網頁「骨架」的核心價值。專案開發過程中，最大的收穫是學會了運用「語意化標籤」來建構清晰的網頁層級，並學會使用瀏覽器開發者工具來進行即時除錯。雖然期間曾因為標籤語法疏忽而導致排版出錯，但透過不斷測試與修正，不僅提升了解決問題的能力，也培養了寫代碼時的嚴謹態度。順利完成期末網頁讓我獲得了極大的成就感，也幫我打下了扎實的前端基礎。未來我將帶著這次的實作經驗，繼續探索 CSS 與 JavaScript 的領域，打造出功能更豐富、視覺更精美的網頁。

李冠勳

網站程式排版設計與修改

本次期末專案是將課堂理論轉化為實務作品的重要里程碑。在獨立開發的過程中，我發現撰寫 HTML 最難的不是死記標籤，而是如何寫出整齊的結構。專案中最大的挑戰在於處理複雜的表單與多媒體物件嵌入，過程中經歷了無數次代碼跑位與除錯的陣痛期。然而，當看到最終網頁能流暢運作、資訊完美呈現時，那份成就感讓所有的熬夜都變得非常值得。這次專案不僅鞏固了我的 HTML 基礎，更讓我磨練出應對程式 bug 的耐心，也讓我對接下來要學習的 CSS 與 JavaScript 充滿期待。

徐儷庭

網站架構草稿與商品內容搜集統整

這次專案讓我體會到資訊系統開發是一個需要跨領域合作的過程，每個人都有不同的分工與專長。未來如果有機會，我希望能進一步學習前端或後端程式設計，提升自己的技術能力，也希望能將這次學到的經驗應用在未來的課程或工作上。整體而言，這次期末專案不僅讓我對商業網站有更完整的認識，也讓我學習到如何與團隊共同完成一項專案，是一次非常寶貴的經驗。
