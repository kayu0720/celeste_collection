退換貨政策

感謝您在 Celeste COLLECTION進行購物。為了保障您的權益，本網站提供完善的售後服務，請在申請退換貨前仔細閱讀以下條款：

1. 七天鑑賞期說明

  • • 非試用期：依《消費者保護法》規定，網購消費者享有商品到貨七天猶豫期（鑑賞期非試用期）。
  • • 計算方式：自您簽收商品（含超商取貨、管理員代收、親友代簽）的隔日起算第一天。
  • • 商品狀態：退回商品必須是全新狀態，且包裝與所有隨附配件、封膜皆完整無缺。

2. 無法辦理退換貨的狀況

  • • 已拆封使用：黑膠唱片外部封膜（塑膠縮膜）已遭拆封、已有聆聽痕跡、外觀刮傷、變形或髒污。
  • • 包裝不完整：黑膠封套、內襯紙套、歌詞本或隨附特典禮品遺失或毀損，或防撕標籤已被拆除。
  • • 超過鑑賞期：自收到商品隔日起算已超過七天。
  • • 除外商品類別：依消保法規定，已拆封之影音軟體（如黑膠唱片、CD、卡帶）屬複製流通販播性質，一經拆封封膜即排除七天猶豫期之適用。

3. 退換貨流程

  • • 聯絡客服：請於收到商品七天內，私訊本站官方 LINE 帳號或寄信至客服信箱提出申請。
  • • 提供資訊：請附上「訂單編號」、「退換貨原因」及「商品外包裝完整封膜與內容物特寫照片」。
  • • 寄回商品：客服人員確認符合資格後，將為您提供專屬退貨代碼或安排精密宅配前往收件。

4. 運費與退款說明

  • • 瑕疵 / 寄錯：若因黑膠唱片本身出廠重大瑕疵（針跳、變形）或我方寄錯商品，退換貨精密物流運費由本網站全額負擔。
  • • 個人因素：若因個人因素（如不喜歡、重複購買）退貨，需由買家自付專用物流運費寄回。
  • • 退款時間：我們在收到退回商品並確認封膜完好無誤後，將於 7 個工作天內完成退款程序。原信用卡付款者將直接辦理線上刷退，轉帳匯款者將退至您指定的實名帳戶。
